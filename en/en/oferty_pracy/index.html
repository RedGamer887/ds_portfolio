Job Offers

📝 App Description

Job Offers is a multilingual Streamlit web application that helps analyze CVs using artificial intelligence (OpenAI GPT-4o) and generates personalized job offers and career advice tailored to your profile and desired position.

💾 Download Notebook + Image 🚀 Try the app

✨ Features

  • 🌐 Multilingual interface (Polish, English, German, Russian)
  • 📄 Automatic CV text analysis from PDF, DOC, DOCX, or TXT files
  • 🈯 Language detection and analysis adaptation
  • 🔍 Extracting and presenting key skills from CVs
  • 📊 Generating matching job offers in a table format (position, company, city, source, description)
  • 💡 Personalized advice for CV improvement, professional growth, and effective job search
  • 🌍 Analysis of external profiles and job offers (LinkedIn, Pracuj.pl, OLX.pl) and candidate matching evaluation
  • 🌈 Clean and user-friendly interface

🛠️ Technologies

  • 🐍 Python 3.11.9
  • 🚀 Streamlit
  • 🤖 OpenAI GPT-4o
  • 📄 PyPDF2 / python-docx / langdetect (file and language analysis)
  • 🗃️ Pandas
